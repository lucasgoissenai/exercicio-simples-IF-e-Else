Variáveis:

let: Utilizamos a palavra-chave let quando queremos criar uma variável que pode ter seu valor alterado ao longo do tempo.

const: A palavra-chave const é usada para criar variáveis com valores fixos ou constantes, que não podem ser alterados após a inicialização.

Estruturas Condicionais:

if: A estrutura if permite a execução de um bloco de código se uma condição especificada for verdadeira.

else: A cláusula else é utilizada para executar um bloco de código quando a condição no if for avaliada como falsa.

else if: A cláusula else if é usada para testar condições adicionais se as condições anteriores no if e else if não forem satisfeitas.

Operadores Matemáticos:

+: O operador de adição (+) é utilizado para somar dois valores.

-: O operador de subtração (-) é usado para subtrair um valor de outro.

*: O operador de multiplicação (*) realiza a multiplicação de dois valores.

/: O operador de divisão (/) divide o primeiro valor pelo segundo.

%: O operador módulo (%) retorna o resto da divisão de um número pelo outro.

Operadores de Comparação:

!=: O operador de não igual (!=) verifica se dois valores são diferentes.

==: O operador de igualdade (==) compara dois valores quanto ao seu conteúdo, ignorando o tipo de dados.

>=: O operador maior ou igual (>=) verifica se um valor é maior ou igual a outro.

<=: O operador menor ou igual (<=) verifica se um valor é menor ou igual a outro.

===: O operador de igualdade estrita (===) compara tanto o valor quanto o tipo de dados dos operandos.

>: O operador maior que (>) compara se um valor é maior que outro.

<: O operador menor que (<) compara se um valor é menor que outro.

Operadores Lógicos:

&& (AND): O operador lógico && retorna verdadeiro apenas se ambos os operandos são verdadeiros.

|| (OR): O operador lógico || retorna verdadeiro se pelo menos um dos operandos é verdadeiro.

! (NOT): O operador de negação (!) inverte o valor de verdadeiro para falso e vice-versa.

Operador de Atribuição:

=: O operador de atribuição (=) é fundamental para atribuir valores a variáveis.
